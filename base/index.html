Architecture

What the Base Subnet does

Miners read, process, and attest to Base chain state. Validators score attestation quality using Resonance BFT. TAO flows to miners who prove their work.

L0 → L1 · Intent
Base Chain Mandates
Validators issue mandates referencing Base block ranges, contract addresses, or onchain events. Miners fetch and process this data as attested mandates.
Base RPC
L2 · Sandbox
Deterministic Execution
Every miner executes in a deterministic sandbox — same Base state in, same output out. Makes attestation proofs reproducible and verifiable by any third party.
Bittensor Axon
L3 · Cognitive Mesh
Base AI Agent Layer
Miners optionally plug in Base's AgentKit — enabling autonomous onchain actions as part of mandate execution. First-class support for CDP SDK and smart wallets.
AgentKit · CDP
L4 · Memory Fabric
Onchain State Index
Miners maintain a local index of Base contract state, transaction history, and event logs. This off-chain memory is referenced in mandate responses and attested to on Bittensor.
Cross-chain
L5 · Action Surface
MCP + Onchain Bindings
Axon endpoints expose Base chain reads as MCP tool bindings — making subnet miners callable by any MCP-compatible AI agent or framework as a live data source.
MCP · A2A · ACP
P3 · Attestation
Merkle-Anchored Proofs
Every Base state claim returned by a miner is Merkle-anchored. Validators verify the proof before assigning weight. Bad data = zero weight = no TAO.
Resonance BFT

# The Base Subnet Synapse — what validators ask, what miners prove
class BaseSubnetSynapse(bt.Synapse):
    # Mandate: what Base data to fetch and attest
    base_block_range: tuple[int, int] = (0, 0)
    contract_address: Optional[str] = None
    query_type: str = ""  # "balance" | "events" | "state" | "agent_action"
    mandate_payload: dict = {}

    # Attestation: Merkle-anchored proof of Base chain state
    base_state_hash: Optional[str] = None
    merkle_proof: Optional[list] = None
    block_hash_anchor: Optional[str] = None  # anchored to Base block hash
    execution_trace: Optional[dict] = None
    confidence: float = 0.0
