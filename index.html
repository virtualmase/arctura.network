Authority flows through the chain, not from it. Each node holds a sealed mandate the network can verify but never override. Agents cannot act beyond their delegated scope.

Byzantine partitions resolve to a single coherent reality. No reorganizations, no rollbacks — only forward attestation. Guaranteed under adversarial conditions.

Provenance is a first-class output. Each action emits a Merkle-anchored proof readable by any verifier on or off-chain. No action without a traceable mandate.
